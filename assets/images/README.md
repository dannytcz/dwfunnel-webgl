# Cyborg hero asset

Drop your AI portrait here:

- **File:** `cyborg-hero.webp` or `cyborg-hero.png`
- **Size:** 2048px tall minimum (portrait 2:3 or 3:4)
- **Background:** solid `#030508` void (no room, no gradient)

The scene auto-loads this path. Until the file exists, the procedural robot placeholder is used.

Optional eye alignment tweak in `hero-scene.js` → `PORTRAIT_CFG` if glow doesn’t sit on her eyes.
